# Native 目录文档

## 概述
`Native` 目录包含了 Gorge Chart Studio 的核心框架和系统组件，这些组件为游戏开发提供了基础功能和抽象。

## 目录结构

- [Framework](Framework.md): 包含核心框架组件，如资源管理、曲线、图形、信号等。
- [System](System.md): 包含系统级组件，如数据结构和系统功能。

## Framework 子目录详细文档

- [Asset](Asset.md): 资源管理组件，如音频、图像和视频资源
- [Curve](Curve.md): 曲线相关组件，包括函数曲线和颜色曲线
- [Deenty](Deenty.md): 音符相关组件
- [Gameplay](Gameplay.md): 游戏玩法相关组件
- [GorgeEditor](GorgeEditor.md): 编辑器相关组件
- [Graphics](Graphics.md): 图形相关组件，如节点、精灵和变换器
- [Signal](Signal.md): 信号相关组件
- [System](System_Framework.md): 系统级工具类，如数学计算、日志记录等
- [Tsiga](Tsiga.md): 自动机和输入处理相关组件

## Framework 根目录文件

- [Framework 根目录文件](FrameworkRoot.md): 核心框架基类，如 Element、Note 等
